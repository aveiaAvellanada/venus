# SP-6: Egresos y Gastos
**Execution Plan**

REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` to implement this plan task-by-task.

## Contexto
Este plan construye el Módulo de Control de Egresos. Permite al dueño registrar todos los gastos de operación (servicios públicos, arriendo, fletes, etc) para poder calcular en el futuro la rentabilidad del negocio.

---

## Task 1: Infraestructura Base de Datos (Storage de Comprobantes)

Necesitamos un baúl (bucket) en la nube para guardar las fotos de las facturas sin mezclarlo con el baúl de "productos".

**Files:** `supabase/migrations/<ts>_sp6_storage_comprobantes.sql`

- [x] **Step 1:** Crear la migración SQL que inserta el bucket `comprobantes` en `storage.buckets`. Como son facturas privadas, el bucket NO debe ser público (`public = false`). Solo usuarios autenticados (`authenticated`) pueden insertar y leer archivos allí.
- [x] **Step 2:** Aplicar migración remotamente (CLI de Supabase vinculada usando Token de proyecto).
- [x] **Step 3:** Commit.
